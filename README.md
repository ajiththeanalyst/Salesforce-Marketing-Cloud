# Salesforce Marketing Cloud (SFMC)

This repository contains my learning notes, hands-on practice, and understanding of **Salesforce Marketing Cloud (SFMC)**.

## What is Salesforce Marketing Cloud?

Salesforce Marketing Cloud (SFMC) is a digital marketing platform used to communicate with customers through multiple channels such as:

* 📧 Email
* 💬 WhatsApp
* 📱 SMS
* 🔔 Push Notifications
* 🌐 Web & Mobile Personalization

Its primary purpose is to deliver the right message to the right audience at the right time through personalized and automated customer journeys.

## Integration with Salesforce CRM

SFMC can be integrated with Salesforce CRM using **Marketing Cloud Connect**.

This integration allows SFMC to:

* Synchronize CRM data such as Contacts, Leads, and Campaigns.
* Use CRM records for marketing communications.
* Trigger customer journeys based on CRM activities.
* Personalize messages using CRM data.

## Major Modules in Salesforce Marketing Cloud

### 1. Contact Builder

* Stores and manages customer data.
* Creates Data Extensions.
* Defines relationships between data.
* Manages Attribute Groups.

### 2. Content Builder

* Creates and manages Emails, SMS, WhatsApp content, Images, and Templates.
* Reuses content across multiple campaigns.

### 3. Automation Studio

* Automates repetitive marketing tasks.
* Imports and exports files.
* Runs SQL Queries.
* Refreshes Data Extensions.
* Schedules automation activities.

### 4. Email Studio

* Creates, tests, and sends marketing emails.
* Manages subscriber lists and email tracking.
* Monitors delivery, opens, clicks, and bounces.

### 5. Journey Builder

* Designs automated customer journeys.
* Sends personalized communications based on customer behavior.
* Supports Email, SMS, WhatsApp, and other communication channels.
